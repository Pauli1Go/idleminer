# Deprecated Assets

These assets belong to the first exploratory asset pass and should not be used for the corrected first-mine layout.

- `assets/environment/background_surface.png`: contains fixed surface structures; use `assets/backgrounds/background_surface_clean.png`.
- `assets/environment/background_underground.png`: too close to a prebuilt tunnel layout; use `assets/backgrounds/background_underground_clean.png`.
- `assets/environment/mine_shaft_frame.png`: unclear placement for the left-elevator/right-shaft layout; use the new mine shaft overlay assets in `assets/world/`.

The older generic sprites in `assets/environment/`, `assets/machines/`, `assets/resources/`, and `assets/ui/` are legacy placeholders. Prefer the corrected `level1` world, character, and coal assets for the playable first-mine scene.
